# Sorteador de Números Aleatórios

## Campos

- **Quantidade de números a serem gerados:** `[Quantidade de números]`
- **Menor número aceitável para o sorteio:** `[Do número]`
- **Maior número aceitável para o sorteio:** `[Até o número]`

## Botões

- **`[Sortear]`**: realiza o sorteio dos números, de acordo com os valores preenchidos nos campos acima.
- **`[Limpar]`**: apaga os valores preenchidos nos campos acima e reinicia o sorteador.

## Resultado

No final da página haverá um texto exibindo os números gerados.
